# m8start

m8 starter project 

## Homepage
- https://m8.nvsx.rocks

## Repositories
- https://github.com/nvsx/m8start
- https://github.com/nvsx/m8

## Setup
- git clone https://github.com/nvsx/m8start
- cd m8start
- cp .env.default .env
- git clone https://github.com/nvsx/m8
- cd m8
- run/liveserver

## Localhost
- Homepage: http://localhost:8088
- Backend: http://localhost:8088/_m8

## Matching m8 version
- v0.0.6
